## Мета роботи: «Сторінка “Світлофор F1”»

Розширити існуючий проєкт «Світлофор», створений під час семестрової роботи, додавши захищену сторінку “Світлофор F1” з окремим компонентом світлофора Формули-1 та його синхронізацією з автомобільним світлофором.

---

## Хід виконання роботи та реалізація вимог

### 1. Роут і сторінка
- **1.1. Створити нову сторінку:** Створено новий React-компонент сторінки `F1TrafficLightPage.jsx`, що містить усю логіку об'єднаних світлофорів.
- **1.2. Додати роут:** У файлі `App.jsx` з використанням React Router додано новий маршрут `/f1-traffic-light`.
- **1.3. Обмежити доступ:** Маршрут захищено за допомогою компонента `ProtectedRoute`, який перевіряє статус авторизації через існуючий механізм перевірки (`AuthContext`). Неавторизовані користувачі не мають доступу до сторінки.

### 2. Розширення існуючого контексту / стану
- **2.1. Додати підтримку F1-світлофора:** Логіку розширено за рахунок локального стану сторінки `F1TrafficLightPage`, де зберігається стан автомобільного світлофора (`isCarGreen`), стан світлофора F1 (`f1State`: "stop" / "start"), та логіка блокування F1-світлофора.
- **2.2. Забезпечити синхронізацію:** Налаштовано узгоджену роботу:
  - Коли автомобільний світлофор показує *зелене світло*, світлофор F1 примусово переходить у стан «Стоп», а кнопка керування F1 повністю блокується.
  - В інших станах автомобільного світлофора (червоний, жовтий) кнопка F1 є активною і виконує свої функції.

### 3. Компонент світлофора Формули-1
- **3.1. Створити новий компонент:** Створено окремий візуальний компонент `F1TrafficLight.jsx`, що приймає пропси для відображення двох станів: «Стоп» (червоне світло) та «Старт» (зелене світло).
- **3.2. Зміна стану:** 
  - *Автоматично:* Налаштовано повзунок для автоматичного перемикання (за замовчуванням кожні 10с) з відображенням прогрес-бару.
  - *Вручну:* Створено інтерактивну кнопку для перемикання користувачем вручну.
- **3.3. Обмеження для кнопки:** Як тільки автомобільний світлофор стає зеленим, кнопка F1-світлофора отримує атрибут `disabled` і його стан примусово стає «Стоп» автоматично.

### 4. Оновлення UI
- **4.1. Відображення світлофорів:** На сторінці `F1TrafficLightPage` чітко відображається автомобільний світлофор (з відліком часу та поточним кольором) та світлофор F1. Кнопка керування F1 має зрозумілий текст, а під час блокування відображається попередження: *"Кнопка заблокована: авто = зелений"*.
- **4.2. Оформлення:** Сторінка має охайний лейаут, розроблений за допомогою Tailwind CSS та DaisyUI, що забезпечує коректне відображення всіх елементів.

### 5. Навігація
- **5.1. Пункт у меню:** В навігаційному барі (Navbar у компоненті `Header`) додано пункт «Світлофор F1», який веде на відповідний маршрут.
- **5.2. Умовне відображення:** Налаштовано так, що посилання на «Світлофор F1» видиме виключно для авторизованих користувачів.

### 6. Публікація та репозиторій
- **6.1. Опублікувати проєкт:** Проєкт з усіма оновленнями підготовлено для розгортання (додатково налаштовано Web Audio API, Google Sheets Stats). Сторінка доступна лише після авторизації.
- **6.2. Запушити роботу:** Код та файли звітів збережені та відправлені у віддалений репозиторій GitHub Classroom. 

### 7. Звіт
- Створено та оформлено даний звіт згідно з екзаменаційними вимогами курсу.
